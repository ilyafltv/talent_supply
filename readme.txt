EN / English

Menu, Footer and Popups - made as js components that are loaded
from the corresponding files in the Components folder.

The page with the choice of services is located in the cabinet.html file, 
after authorization you will need to add another section to the menu 
- Cabinet(cabinet.html), Filter functionality is implemented
in scripts/cabinet.js

Slider functionality (from the main page), footer arrows and burger menu 
implemented in scripts/script.js

RU / Русский

Меню, Футер и Попапы - сделаны как js-компоненты, которые подгружаются
из соответствующих файлов папки Components.

Страница с выбором услуг находится в файле cabinet.html, после авторизации
нужно будет добавлять в меню еще один раздел - Cabinet(cabinet.html),
Функционал фильтров реалзиован в scripts/cabinet.js

Функционал слайдера(с главной страницы), стрелочки из футера и бургер-меню 
реализован в scripts/script.js
